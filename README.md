# LR Digital — Automação e IA Comercial

Landing page institucional para divulgação do serviço de consultoria e implementação de automação comercial com IA da LR Digital.

## Tecnologias
- React
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Lucide React

## Como rodar localmente

```bash
npm install
npm run dev
```
